PCB WorkFlow Tools

PDF → Altium Pin Table

Upload a datasheet pin-definition PDF (Espressif ESP32 or STM32 pinout tables). Get separate columns for Designator, Display Name, and Electrical Type to paste into Altium’s Symbol Wizard. Processing happens entirely in your browser.

1. Upload PDF

Detected from PDF when you choose a file
e.g. GPIO1/RTC_GPIO1 or GPIO1-RTC_GPIO1

Choose a PDF, then click Convert.

2. Copy columns

Use Copy column (or select all in a box) and paste into the matching Symbol Wizard field.

Results appear here after conversion.

1. Upload PCB image

Works best with a top-down photo or render on a contrasting background. Crop to the board if possible.

Altium import: File → Import → DXF/DWG → choose Model space → set 1 AutoCAD unit = 1 mil (or 1 mm if you exported millimeters). Match the DXF units dropdown above.

Choose an image, then click Convert outline.
